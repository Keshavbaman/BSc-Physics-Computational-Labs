# BSc Physics Computational Labs 

## Overview
This repository serves as a comprehensive portfolio of computational physics projects completed during my BSc Physics Honors at **Miranda House, University of Delhi**. It covers numerical analysis, simulation of physical systems, and quantum mechanical calculations using Python.

## Repository Structure & Syllabus Mapping

### 1. Mathematical Physics I (Semester 1)
**Focus:** Basics of Python, Root Finding, and Integration.
* `Sem1_MP1/basics_plotting.py`: Visualization of mathematical functions.
* `Sem1_MP1/newton_raphson.py`: Root finding algorithm.
* `Sem1_MP1/simpsons_rule.py`: Numerical integration (Simpson's 1/3 Rule).
* `Sem1_MP1/euler_method.py`: First-order ODE solver (Radioactive Decay).

### 2. Mathematical Physics II (Semester 2)
**Focus:** Curve Fitting, Linear Algebra, and Differential Equations.
* `Sem2_MP2/curve_fitting.py`: Least Square Fit for Ohm's Law.
* `Sem2_MP2/gauss_elimination.py`: Solving linear systems.
* `Sem2_MP2/harmonic_oscillator.py`: RK4 solution for Damped Harmonic Oscillator.

### 3. Mathematical Physics III (Semester 3)
**Focus:** Fourier Analysis and Special Functions.
* `Sem3_MP3/fourier_square_wave.py`: Fourier Series summation.
* `Sem3_MP3/dirac_delta.py`: Numerical limit verification of the Delta function.
* `Sem3_MP3/fft_transform.py`: Fast Fourier Transform (FFT) of Gaussian.

### 4. Quantum Mechanics (Semester 5)
**Focus:** Solving the Schr√∂dinger Equation (Shooting Method).
* `Sem5_Quantum_Mechanics/hydrogen_atom.py`: Ground state energy of Hydrogen (-13.6 eV).
* `Sem5_Quantum_Mechanics/anharmonic_oscillator.py`: Nuclear potential simulations ($V \propto r^2 + r^3$).

## Technology Stack
* **Language:** Python 3
* **Libraries:** `numpy`, `scipy`, `matplotlib`
* **Methods:** Newton-Raphson, Simpson's Rule, Runge-Kutta (RK4), Shooting Method.
